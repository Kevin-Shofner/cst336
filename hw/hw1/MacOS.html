MacOS is a OS that is primarily used for Mac computers, and is developed by Apple.

The current version of MacOs is High Sierra. Some of the features that are exclusive to Mac are Siri,

or Apple's version of storing things online on called Icloud. If you want to use a Macbook, you will end up

using MacOS as it's OS.
