Windows is a OS that is primarily used for Windows computers, and is developed by Microsoft.

The current version of Windows is Windows 10. Some of the features that are exclusive to Windows are

Cortana, being the most supported OS computer to play video games on, or being able to use Microsoft

Office. If you end up buying a computer from Microsoft, it will use the Windows OS.
